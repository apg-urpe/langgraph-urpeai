## Tool: getConversationalIntelligence

### Implementación
- **Archivo modificado**: `app/api/chat/route.ts`
- **Tool agregada**: `getConversationalIntelligence` (TOOL 5)
- **Edge Function**: `https://vecspltvmyopwbjzerow.supabase.co/functions/v1/-obtener_mensajes_conversaciones`

### Funcionalidad
Permite a Monica analizar bloques de conversaciones con mensajes RAW para:
- Detectar patrones de abandono
- Identificar tendencias de comunicación
- Análisis cualitativo de big data conversacional
- Evaluar calidad de respuestas del agente

### Parámetros
| Parámetro | Tipo | Descripción |
|-----------|------|-------------|
| start_date | string | Fecha inicio YYYY-MM-DD (requerido) |
| end_date | string | Fecha fin YYYY-MM-DD (requerido) |
| ordenar_por | enum | created_at \| updated_at (default: updated_at) |
| orden | enum | desc \| asc (default: desc) |
| limite | number | 1-500 conversaciones (default: 100) |

### Tools Añadidas

#### 1. `webSearch` - Búsqueda en Internet
```typescript
webSearch({
  query: "mejores prácticas CRM 2024",
  limit: 5,              // default: 5, max: 10
  scrapeResults: false   // true = extrae contenido completo
})
```

#### 2. `webScrape` - Scraping de URLs
```typescript
webScrape({
  url: "https://example.com/page",
  onlyMainContent: true,  // default: true
  includeLinks: false     // default: false
})
```

### Configuración
- **API Key**: `FIRECRAWL_API_KEY` (ya existe en `.env`)
- **API URL**: `https://api.firecrawl.dev/v1`
- **Condicional**: Las tools solo se cargan si `FIRECRAWL_API_KEY` está configurada

### Límites y Truncado
- `webSearch` resultados: 3,000 chars/resultado
- `webScrape` contenido: 15,000 chars máximo
- Rate limits manejados automáticamente (429)
- Créditos agotados manejados (402)

### Uso Esperado
- "Busca en internet sobre X"
- "Investiga qué dice la web sobre nuestra competencia"
- "Extrae el contenido de esta URL"
- "Lee esta página web"
