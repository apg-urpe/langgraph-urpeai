## Sistema de Historial de Cambios - Empresa (15 Ene 2026)

### Objetivo
Implementar un sistema de auditoría para rastrear cambios en la configuración de empresa (`wp_empresa_perfil`), similar al existente para agentes.

### Arquitectura

El sistema usa la tabla `wp_auditoria` existente con una vista especializada:

```
wp_empresa_perfil
       │
       ▼ (trigger UPDATE)
wp_auditoria (tabla = 'wp_empresa_perfil')
       │
       ▼ (vista)
wp_empresa_historial
```

### Schema SQL

#### Vista: `wp_empresa_historial`
```sql
CREATE VIEW wp_empresa_historial AS
SELECT 
  a.id,
  a.registro_id AS empresa_id,
  a.campo,
  a.valor_anterior,
  a.valor_nuevo,
  a.usuario_id,
  a.mensaje_commit,
  a.fecha AS created_at
FROM wp_auditoria a
WHERE a.tabla = 'wp_empresa_perfil';
```

#### Campos Auditados
- **Identidad**: nombre, ciudad, pais, rubro
- **Contenido**: informacion_empresarial, preguntas_frecuentes, servicios_generales, embudo_ventas
- **Contacto**: logo_url, sitio_web, telefono, email, direccion
- **Configuración**: reglas_negocio, canal_comunicacion, timezone, team_slack
- **Metadata**: metadata, branding
- **Flags**: activo, metricas_activa, email_marketing

#### Funciones RPC

| Función | Propósito |
|---------|-----------|
| `fn_get_enterprise_history(empresa_id, campo?, limit?)` | Obtener historial filtrado |
| `fn_restore_enterprise_field(historial_id, usuario_id?)` | Restaurar valor anterior |

### Componente UI: EnterpriseHistoryViewer

**Características:**
- Modal fullscreen con listado de cambios
- Filtros por campo modificado
- Expandir/colapsar entradas
- Diff visual (valor anterior en rojo, nuevo en verde)
- Botón "Restaurar valor anterior" por entrada
- Información del usuario que hizo el cambio
- Timestamps relativos (hace X min/horas/días)

### Integración en SettingsView

Nuevo botón con icono `History` en el header de Configuración de Empresa:
- Ubicado junto a "Refrescar" y "Guardar"
- Abre el modal `EnterpriseHistoryViewer`
- Disponible para todos los roles (lectura)

### Uso

```sql
-- Consultar historial de empresa 13
SELECT * FROM wp_empresa_historial WHERE empresa_id = 13;

-- O usar la función RPC
SELECT * FROM fn_get_enterprise_history(13);
SELECT * FROM fn_get_enterprise_history(13, 'informacion_empresarial');

-- Restaurar un valor anterior
SELECT fn_restore_enterprise_field(historial_id, usuario_id);
```
