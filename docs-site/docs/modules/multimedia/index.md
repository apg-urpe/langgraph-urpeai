# Multimedia - Archivos en el Chat

> Comparte archivos con Monica AI para analisis inteligente de imagenes, documentos, audio y video

---

## Proposito

El modulo multimedia permite a los usuarios enviar archivos directamente en el chat con Monica AI. La inteligencia artificial puede analizar, describir, resumir y responder preguntas sobre cualquier tipo de archivo soportado.

---

## Tipos de Archivo Soportados

### Imagenes

- JPEG, PNG, GIF, WebP

### Documentos

- PDF

### Audio

- MP3, WAV, AAC, OGG, FLAC, WebM Audio

### Video

- MP4, MPEG, MOV, AVI, WebM Video

---

## Limites

| Parametro | Valor |
|-----------|-------|
| Tamano maximo por archivo | 20 MB (50 MB para video) |
| Archivos por mensaje | 5 maximo |
| Resolucion de imagenes | Se redimensiona automaticamente a 1024px max |
| Compresion de imagenes | Compresion automatica para optimizar envio |

---

## Formas de Agregar Archivos

| Metodo | Descripcion |
|--------|-------------|
| Boton de clip | Abre el selector de archivos del sistema |
| Arrastrar y soltar | Arrastra archivos directamente sobre el area de chat |
| Pegar (Ctrl+V) | Pega imagenes del portapapeles directamente |

---

## Vista Previa de Archivos

- **Imagenes**: Miniatura visual antes de enviar
- **Otros archivos**: Tarjeta con icono del tipo de archivo y nombre

---

## Ejemplos de Uso con Monica AI

Monica puede analizar cualquier archivo soportado. Algunos ejemplos:

- **Imagen**: "Que ves en esta imagen?" - Monica describe el contenido visual
- **PDF**: "Resume este documento" - Monica extrae los puntos principales
- **Audio**: "Transcribe este audio" - Monica convierte el audio a texto
- **Video**: "Que sucede en este video?" - Monica analiza el contenido del video

---

## Seguridad

- Solo usuarios autenticados pueden subir archivos
- Cada archivo queda asociado al usuario y la sesion de chat
- Solo el propietario puede eliminar sus archivos
