# Menciones (@usuario)

> Notificaciones contextuales para colaboracion entre miembros del equipo

---

## Proposito

El sistema de menciones permite a los usuarios notificar a otros miembros del equipo escribiendo `@nombre` en comentarios, notas y otros campos de texto. Esto facilita la colaboracion y asegura que las personas correctas sean informadas sobre conversaciones relevantes.

---

## Donde se Pueden Usar Menciones

| Contexto | Descripcion |
|----------|-------------|
| Comentarios de tareas | Mencionar a un companero en una tarea |
| Notas de contacto | Alertar sobre informacion importante de un cliente |
| Notas de citas | Coordinar acciones antes o despues de una reunion |
| Actualizaciones de proyecto | Informar avances a miembros especificos |

---

## Como Funciona

1. **Escribir @** en cualquier campo de texto compatible
2. **Autocompletado** muestra una lista de miembros del equipo que coinciden
3. **Seleccionar** al usuario deseado de la lista
4. **Al enviar**, el sistema crea automaticamente una notificacion para cada persona mencionada

---

## Notificaciones de Menciones

Cuando alguien te menciona:

- Notificacion en tiempo real dentro de la plataforma
- Badge en el Centro de Actividad
- Click en la notificacion te lleva directamente al contexto (tarea, nota, etc.)

---

## Reglas

| Regla | Descripcion |
|-------|-------------|
| No auto-mencion | No puedes mencionarte a ti mismo |
| Solo equipo activo | Solo se pueden mencionar usuarios activos de la empresa |
| Limite por mensaje | Maximo 10 menciones por mensaje |
| Acceso validado | Solo se notifica si el mencionado tiene acceso al contexto |
