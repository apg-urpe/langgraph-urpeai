# Novedades

> Visualizacion de actualizaciones y mejoras de la plataforma

---

## Proposito

El sistema de novedades permite comunicar a los usuarios las ultimas funciones, mejoras y correcciones implementadas en Urpe AI Lab. Los usuarios siempre estan informados sobre que hay de nuevo en la plataforma.

---

## Ubicacion

Las novedades estan integradas dentro del **Centro de Actividad** en el header del panel de administracion, junto con las notificaciones. Se accede mediante la pestana "Novedades" en el dropdown unificado.

---

## Funcionalidades

| Funcion | Descripcion |
|---------|-------------|
| Badge de novedad | Indicador visual (punto azul) cuando hay actualizaciones no vistas |
| Categorizacion | Las novedades se organizan en: Funciones Listas, Mejoras, Proximamente |
| Persistencia | El sistema recuerda cuando fue la ultima vez que revisaste las novedades |
| Interfaz unificada | Integrado en el mismo dropdown que las notificaciones |

---

## Como Funciona

1. Cuando se publican nuevas actualizaciones, aparece un **badge azul** en el boton del Centro de Actividad
2. Al abrir el Centro de Actividad, selecciona la pestana **"Novedades"**
3. Se muestran las ultimas funciones y mejoras organizadas por categoria
4. El badge desaparece automaticamente hasta la proxima actualizacion

---

## Categorias de Novedades

| Categoria | Descripcion |
|-----------|-------------|
| Funciones Listas | Nuevas funcionalidades ya disponibles para usar |
| Mejoras Importantes | Optimizaciones y mejoras a funciones existentes |
| Proximamente | Funcionalidades en desarrollo que estaran disponibles pronto |
