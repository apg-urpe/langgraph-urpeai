## Que es

Monica Chat es donde tu equipo trabaja con IA. No es un buscador — es una asistente que entiende tu negocio, ejecuta acciones y genera respuestas visuales con los datos reales de tu CRM.

![Vista principal de Monica Chat](/screenshots/chat/chat-main-view.png)

---

## Que puedes hacer

### Consultar tu negocio en lenguaje natural

No necesitas dashboards ni reportes. Pregunta directamente:

| Tu preguntas | Monica hace |
|-------------|-------------|
| *"Cuantos leads entraron esta semana?"* | Consulta el CRM, cuenta y te da el numero con contexto |
| *"Quien es mi mejor prospecto?"* | Calcula lead scoring, ordena y te muestra el top |
| *"Como va el equipo de ventas?"* | Accede al dashboard, genera KPIs y graficos |
| *"Que citas tengo manana?"* | Revisa tu calendario y lista las citas con detalles |
| *"Cuanto nos deben en cartera?"* | Suma saldos pendientes de todos los servicios |

### Gestionar el CRM sin salir del chat

| Tu pides | Monica ejecuta |
|----------|----------------|
| *"Agrega una nota al contacto Maria Garcia"* | Crea la nota en el perfil del contacto |
| *"Muevelo a la etapa de negociacion"* | Actualiza el pipeline del contacto |
| *"Crea una tarea de seguimiento para el lunes"* | Genera la tarea asignada a ti |
| *"Que notas tiene el contacto 573001234567?"* | Lee todas las notas del perfil |

### Analizar archivos

Envia un archivo y Monica lo procesa:

| Envias | Monica responde |
|--------|-----------------|
| Foto de un comprobante | Extrae monto, fecha y datos del pago |
| PDF de un contrato | Resume las clausulas principales |
| Audio de una llamada | Transcribe y analiza el sentimiento |
| Captura de pantalla | Describe lo que ve y responde preguntas |

### Investigar en la web

| Tu pides | Monica hace |
|----------|-------------|
| *"Investiga a la empresa XYZ"* | Lanza Deep Research, busca en la web y entrega un reporte |
| *"Que hace la competencia en este sector?"* | Investigacion profunda con fuentes |

## Herramientas

Monica tiene 20+ herramientas que ejecuta automaticamente. No necesitas saber cuales son — ella decide cual usar segun lo que necesitas.

### CRM & Contactos

| Herramienta | Cuando la usa |
|-------------|---------------|
| **Buscar contactos** | "Busca a...", "Cuantos contactos...", "Quien tiene..." |
| **Perfil completo** | "Dime todo sobre...", "Que sabemos de..." |
| **Lead scoring** | "Mejores leads", "Quien tiene mas potencial" |
| **Notas** | "Agrega una nota", "Que notas tiene..." |
| **Contexto 360** | Analisis profundo: conversaciones + citas + notas + tareas |

### Pipeline & Ventas

| Herramienta | Cuando la usa |
|-------------|---------------|
| **Etapas del funnel** | "Como va el pipeline?", "Cuantos en negociacion?" |
| **Estadisticas** | "Tasa de conversion", "Contactos por etapa" |
| **Proyectos** | "Que proyectos tenemos?", "Costos del proyecto X" |

### Metricas & Reportes

| Herramienta | Cuando la usa |
|-------------|---------------|
| **KPIs del negocio** | "Como vamos este mes?", "Metricas del equipo" |
| **Campanas** | "Resultados de la campana X", "Tasa de apertura" |
| **Equipo** | "Rendimiento por asesor", "Quien tiene mas citas?" |

### Calendario & Tareas

| Herramienta | Cuando la usa |
|-------------|---------------|
| **Citas** | "Citas de hoy", "Agenda de la semana" |
| **Tareas** | "Tareas pendientes", "Que hay por hacer?" |

### Busqueda Avanzada

| Herramienta | Cuando la usa |
|-------------|---------------|
| **Sub-agente CRM** | Preguntas complejas que requieren cruzar multiples datos |

Monica delega al sub-agente cuando necesita combinar informacion de contactos, citas, campanas y finanzas en una sola respuesta.

## Respuestas visuales

Monica no solo responde con texto. Genera bloques visuales:

| Bloque | Cuando aparece |
|--------|----------------|
| **KPIs** | Tarjetas con numeros grandes y tendencias |
| **Tablas** | Listas de contactos, citas, tareas |
| **Graficos** | Barras, lineas, pie charts con tus datos |
| **Formularios** | Para crear contactos o tareas desde el chat |
| **Cards** | Resumen visual de un contacto o servicio |
