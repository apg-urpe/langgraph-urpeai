## Que es

El modulo de Contactos es el corazon de tu CRM. Aqui gestionas todas las relaciones comerciales de tu empresa:

- **Captura de leads** desde WhatsApp, web y entrada manual
- **Cualificacion automatica** basada en conversaciones y actividad
- **Pipeline visual** con embudo configurable por etapas
- **Historial completo** de cada contacto: conversaciones, citas, notas, archivos, tareas y campanas
- **Busqueda avanzada** con lead scoring inteligente

---

## Panel de Detalle

Cada contacto tiene un panel completo con toda su informacion organizada en tabs:

| Tab | Que muestra |
|-----|-------------|
| **Conversaciones** | Historial completo de chats de WhatsApp y web |
| **Citas** | Todas las citas programadas, realizadas y canceladas |
| **Multimedia** | Archivos compartidos (imagenes, PDFs, documentos) |
| **Notas** | Base de conocimiento del contacto con tags |
| **Tareas** | Tareas vinculadas al contacto |
| **Marketing** | Campanas de email en las que esta inscrito |
| **Monica AI** | Chat contextual — Monica ya conoce todo sobre este contacto |

---

## Busqueda Avanzada

El sistema tiene tres niveles de busqueda segun la profundidad que necesites:

| Nivel | Velocidad | Donde busca |
|-------|-----------|-------------|
| **Local** | Instantanea | Filtra en los contactos ya cargados |
| **Basica** | Rapida | Nombre, telefono, email en la base de datos |
| **Super Search** | Completa | Todo: contactos + mensajes + notas + metadata |

Puedes buscar por nombre, telefono, email, contenido de mensajes, notas o cualquier campo personalizado.

---

## Sistema de Pausa

A veces necesitas pausar temporalmente la gestion de un contacto:

| Estado | Que significa | Indicador |
|--------|---------------|-----------|
| **Activo** | Gestion normal | Verde |
| **Pausado** | Pausa temporal (5, 15, 30 min o permanente) | Ambar con cuenta regresiva |
| **Desactivado** | Sin gestion activa | Rojo |

La pausa es util cuando un contacto pidio que no lo contacten por un tiempo o cuando estas esperando informacion.

---

## Lead Scoring

El sistema calcula automaticamente una puntuacion de 0 a 100 para cada contacto basada en:

- **Cualificacion** — si ya fue marcado como calificado (+30 puntos)
- **Estado** — si ya es cliente (+20 puntos)
- **Actividad reciente** — interaccion en las ultimas 24 horas (+25 puntos)
- **Tags especiales** — contactos VIP o con tags prioritarios (+15 puntos)

El lead scoring ayuda a tu equipo a priorizar: los contactos con mayor puntuacion son los que tienen mas probabilidad de conversion.
