## Como se Conectan

Todos los modulos comparten datos a traves de:

1. **Contactos como eje central** вҖ” cada conversacion, cita, tarea, pago y campana esta vinculada a un contacto
2. **Filtro global de equipo** вҖ” filtra todos los modulos por miembro del equipo
3. **Contexto de empresa** вҖ” cada empresa ve solo sus propios datos (multi-tenant)
4. **Monica IA** вҖ” tiene acceso de lectura a todos los modulos para responder consultas
