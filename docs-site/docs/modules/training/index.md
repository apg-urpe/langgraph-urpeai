# Urpe Academy

> Plataforma de capacitacion gamificada para equipos comerciales

---

## Proposito

Urpe Academy es un modulo de capacitacion inspirado en Duolingo, disenado para entrenar equipos comerciales usando contenido generado por Monica AI basado en el contexto real del CRM de cada empresa.

---

## Caracteristicas Principales

- **Modo Enfoque**: Pantalla completa que elimina distracciones durante el entrenamiento
- **Generacion con IA**: Los cursos se crean automaticamente desde datos reales del CRM
- **Gamificacion**: Rachas, XP, corazones (vidas) y tablas de clasificacion
- **Sesiones interrumpibles**: Puedes pausar y retomar donde lo dejaste
- **Multi-empresa**: Cada empresa crea y gestiona sus propios cursos

---

## Estructura de Contenido

### Cursos

Cada empresa puede crear cursos de capacitacion organizados por:

| Campo | Descripcion |
|-------|-------------|
| Titulo | Nombre del curso |
| Categoria | Tema del curso (ventas, servicio, producto, etc.) |
| Dificultad | Principiante, Intermedio o Avanzado |
| Duracion estimada | Tiempo aproximado para completar el curso |

### Lecciones

Cada curso contiene lecciones ordenadas secuencialmente. Cada leccion incluye:

- Introduccion al tema
- Conjunto de preguntas interactivas
- Recompensa de XP al completar

### Tipos de Preguntas

| Tipo | Descripcion |
|------|-------------|
| Opcion multiple | 4 opciones, 1 correcta |
| Verdadero/Falso | Evalua afirmaciones |
| Ordenar pasos | Ordena una secuencia logica |
| Completar oracion | Rellena el espacio en blanco |
| Roleplay | Escenario conversacional simulado |

Las preguntas pueden ser creadas manualmente o generadas automaticamente por Monica AI usando el contexto del CRM (productos, objeciones comunes, casos de exito).

## Mecanicas de Gamificacion

### Sistema de Corazones (Vidas)

| Accion | Resultado |
|--------|-----------|
| Respuesta correcta | Mantiene los corazones |
| Respuesta incorrecta | Pierde 1 corazon |
| 0 corazones | Leccion fallida - esperar o practicar mas |

### Sistema de XP (Experiencia)

| Accion | XP |
|--------|-----|
| Pregunta correcta | +10 XP |
| Leccion completada | +50 XP bonus |
| Sin errores (Perfecto) | +25 XP bonus |
| Racha activa | x1.5 multiplicador |

Los puntos de XP se integran con el sistema de gamificacion general de la plataforma, contribuyendo a badges y tablas de clasificacion.

---

## Experiencia de Entrenamiento

### Modo Enfoque

Al iniciar una leccion, se activa el modo enfoque que ocupa toda la pantalla:

- Barra de progreso mostrando avance en la leccion
- Indicadores de corazones restantes y XP acumulado
- Tecla ESC para pausar (con confirmacion)
- Animaciones visuales para respuestas correctas e incorrectas

### Estados Visuales de Respuesta

| Estado | Efecto Visual |
|--------|---------------|
| Sin responder | Opciones en color neutro |
| Seleccionada | Opcion resaltada |
| Correcta | Verde + animacion de celebracion + XP |
| Incorrecta | Rojo + sacudida + -1 corazon |
| Leccion completa | Pantalla de resumen con puntaje total |

## Metricas de Exito

| Metrica | Objetivo |
|---------|----------|
| Tasa de completado | Mas del 70% de lecciones completadas |
| Usuarios activos diarios | Incremento del 25% en uso |
| Retencion de conocimiento | Score promedio mayor al 60% en evaluaciones |
| Tiempo por leccion | Menos de 5 minutos promedio |
