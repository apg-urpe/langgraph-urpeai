# v4.5.2 — Limpieza del sistema de versionado

> Fecha: 2026-03-30 | Rama: panel-v2 | Riesgo maximo: MEDIO

## Resumen

Se elimina por completo el sistema legacy de version checking, changelog parser y notificaciones de actualizacion. Estos mecanismos no aportaban valor real y agregaban complejidad al build y al layout. El panel de notificaciones se simplifica a una sola vista limpia.

## Cambios

### refactor
- Eliminado sistema de version checking: `useVersionChecker`, `versionChecker`, `changelogParser`, `AppUpdatesModal`, `VersionUpdateNotification` (UI)
- Simplificado `NotificationDropdown` eliminando tab de "Novedades"/changelog (UI)
- Removido `NotificationButton.hasNewChangelog` prop (UI)
- Removida logica de changelog badge en `AdminPanel` (UI)

### chore
- Eliminados scripts `inject-version.js` y `version-inject.js` del build pipeline (CONFIG)
- Simplificados scripts `dev` y `build` en package.json (CONFIG)
- Removida ruta `/api/version` del middleware y del API (API, AUTH)
- Eliminado `public/CHANGELOG.md` (DOCS)

### docs
- Reescrita documentacion de Monica Chat con escenarios reales y screenshots (DOCS)
- Reescrita documentacion de Monica IA con casos de uso por rol (DOCS)
