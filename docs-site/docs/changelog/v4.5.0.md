# v4.5.0 — Plantillas WhatsApp desde Monica

> Fecha: 2026-03-28 | Rama: panel-v2 | Riesgo maximo: ALTO

## Resumen

Monica ahora puede crear borradores de plantillas WhatsApp directamente desde el chat. El usuario describe lo que necesita y Monica genera la plantilla con variables, la guarda como borrador y entrega un link navegable para revisarla en Ajustes. Tambien se reescribio la documentacion de integraciones con un enfoque mas claro para el equipo.

## Cambios

### feat
- Nuevo tool `createTemplateDraft` en Monica — crea borradores de plantillas WhatsApp desde el chat (API)
- Links clickeables `template://ID` en el chat para navegar a la plantilla creada (UI)
- Navegacion `focusTemplateNavigation` en admin store para abrir la vista de plantillas (STATE)

### docs
- Reescritura completa de documentacion de integraciones: Supabase, Gemini, Nylas, OpenRouter, MCP Tools (DOCS)
- Simplificacion del indice de integraciones y guia de setup (DOCS)
