# v4.0.0 — Baseline: Monica AI con Tools

> Fecha: 2026-03-28 | Rama: panel-v2 | Version base documentada

## Resumen

Version base del proyecto con toda la infraestructura de Monica AI establecida. Este release documenta el estado del sistema al momento de introducir el sistema de changelog automatizado.

## Capacidades incluidas

### Monica AI
- 39 tools integrados via Gemini 3 Flash con function calling
- Sistema de roles personalizables (database-backed)
- Multi-step tool calling (hasta 15 iteraciones)
- Fallback automatico a OpenRouter
- Streaming via UI Message Protocol

### Modulos CRM
- Contactos con busqueda, perfil 360, notas, asignaciones
- Pipeline/Funnel con etapas y estadisticas
- Portfolio financiero con pagos, ledger, cola de cobro
- Calendario via Nylas (crear, editar, eliminar eventos)
- Email via Nylas (buscar, leer correos)
- Tareas con proyectos y checklists
- Equipo con miembros y asignaciones

### Integraciones
- Supabase (DB, Auth, Storage, RLS)
- Google Gemini 3 Flash (AI principal)
- OpenRouter (fallback AI)
- Nylas (calendario + email)
- Firecrawl (web search + scraping)
- E2B (ejecucion de Python)

### Infraestructura
- Next.js 14 desplegado en Vercel
- Supabase proyecto: Monica CRM Inteligent
- Multi-tenant con aislamiento por empresa_id
