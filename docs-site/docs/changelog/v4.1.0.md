# v4.1.0 — Documentacion inteligente y tooling

> Fecha: 2026-03-28 | Rama: panel-v2 | Riesgo maximo: ALTO

## Resumen

Monica ahora puede consultar la documentacion tecnica de la plataforma para guiar a los usuarios. Se despliega el sitio de documentacion en Vercel y se introduce el sistema de changelog automatizado.

## Cambios

### feat
- Nuevo tool `searchDocumentation` para Monica con modos search y read
- Build-time docs indexer (108 docs indexados en JSON para compatibilidad serverless)
- MCP server `urpe-docs` para consulta de documentacion desde agentes
- Docs site VitePress desplegado en Vercel
- Sistema de changelog automatizado en /push skill
- Skill `/supabase` para gestion segura de la BD de produccion

### fix
- Sanitizacion de PII en documentacion (emails, telefonos, secrets reales removidos)
- Routing de VitePress corregido (README.md → index.md, cleanUrls habilitado)
- Rotacion recomendada de secret `X_URPE_AUTH` expuesto

### chore
- Configuracion de Vercel CLI + GitHub CLI
- Vinculacion de proyecto Supabase (Monica CRM Inteligent)
- Vinculacion de proyecto Vercel (urpe-ai-lab/chat-urpe-ai-lab-1-1)
