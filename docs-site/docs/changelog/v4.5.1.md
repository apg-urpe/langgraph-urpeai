# v4.5.1 — Navegacion a plantillas + docs multicanal

> Fecha: 2026-03-30 | Rama: panel-v2 | Riesgo maximo: MEDIO

## Resumen

Completa el flujo de creacion de plantillas desde Monica: al hacer clic en el link `template://ID` en el chat, la app navega automaticamente a Ajustes > Plantillas WhatsApp y abre el borrador para editarlo. Ademas, se actualizo la documentacion publica con enfoque multicanal (7 canales) y busqueda en espanol.

## Cambios

### feat
- Navegacion automatica a plantilla creada desde chat: SettingsView auto-navega al tab WhatsApp y WhatsAppTemplatesSection auto-abre el modal del borrador (UI)

### docs
- Reescritura de overview, getting-started, chat audit y Monica AI docs con enfoque multicanal (DOCS)
- Nav del docs site renombrado: Plataforma, Comenzar, Producto, Canales, Novedades (DOCS)
- Busqueda local traducida al espanol (DOCS)
- TeamHome actualizado con 7 canales y textos simplificados (DOCS)
