# Dashboard Module

## Descripción
Módulo del Dashboard de Métricas con visualización de KPIs en tiempo real.

## Componentes

### DashboardSkeleton.tsx
Skeleton de carga para el dashboard.

### DateRangeFilter.tsx
Filtro de periodo temporal con opciones predefinidas y rango personalizado.

## Dependencias
- `DashboardView.tsx` (padre en `components/admin/`)
- `useAdminMetrics` hook para datos
- `SafeBlockRenderer` para renderizado de bloques
